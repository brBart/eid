eID command line utilities examples / käsurea koodinäited
==========================================================

Code examples for using Estonian ID-card Java and C libraries.

Koodinäited Eesti ID-kaardi Java ning C teekide kasutamiseks.

Vaata ülevaadet, konfigureerimise ja kompileerimise juhendeid
lehelt

http://eid.eesti.ee/index.php/K%C3%A4surea_n%C3%A4iterakendused

Sisu:

* Java/JavaSign.java: allkirjastamise näide jdigidoc teegi vastu
* Java/JavaAuth.java: autoriseerimise näide jdigidoc teegi vastu
* C/common/ : vajalikud osad nii auth.cpp kui sign.cpp jaoks
* C/sign.cpp,auth.cpp,decrypt.cpp, verify.cpp: näited libdigidocpp vastu

mobile ID authentication / mobiil-ID isikutuvastus
===================================================

Example for using Estonian mobile-ID for authentication:
source from the page http://demo.digidoc.ee/auth_sample/
unpacked and unmodified. The backgrounds specs are here:
http://www.sk.ee/upload/files/DigiDocService_spec_est.pdf
